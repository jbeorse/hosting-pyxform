XLS2XForm

XLS2XForm is a tool to simplify the creation of forms. Forms can be designed with Excel and XLS2XForm will convert it to a XForm that can be used with ODK tools.

Using the Application

  1. Please read all the instructions and notes before beginning.
  2. Design your form following this tutorial or using this XLS sample.
  3. Choose your .xls file and submit it for conversion.

  4. A Download XForm button will appear if the conversion is successful.
